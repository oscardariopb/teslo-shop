# Teslo API

1. Clonar proyecto
2. yarn install
3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Cambiar variables de entorno
5. Levantar la base de datos

```
docker compose up -d
```

6. Ejecutar Seed

```
GET localhost:3000/api/seed
```

7. Levantar la app

```
yarn start:dev
```
